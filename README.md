#   有券后台管理系统

## 项目目录结构
```
- config
    - config.js  // umijs的总配置文件
    - plugin.config.js  // webpack相关配置文件
    - router.config.js  // 配置式路由表
- mock  // 本地mock
- src   // 项目目录
    - assets
    - components            // 通用主键
    - layouts               // 通用大布局
    - models                // 通用model
    - pages                 // 页面
    - services              // api
    - utils                 // 工具类库
    app.js                  // 配置dva钩子 等
    defaultSettings.js      // 主题默认样式
